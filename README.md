# 🎯 Simulación de Círculos en Canvas

Aplicación web interactiva desarrollada con **HTML, CSS, JavaScript y Bootstrap**, que permite visualizar diferentes comportamientos de círculos dentro de un canvas.

---

## 🚀 Características

* 🔵 **Movimiento:**
  Los círculos se desplazan aleatoriamente y rebotan en los bordes del canvas.

* 🟡 **Colisión:**
  Detecta cuando los círculos se tocan y cambian de color sin modificar su trayectoria.

* 🔴 **Rebote:**
  Simulación de colisiones físicas donde los círculos rebotan entre sí y cambian de color.

* 🎛 **Control dinámico:**
  Un slider permite modificar la cantidad de círculos en tiempo real.

* 💎 **Diseño moderno:**
  Interfaz con estilo *glassmorphism*, fondo personalizado y tipografía elegante.

---

## 🧠 Tecnologías utilizadas

* HTML5

## 🎮 Uso

* Usa el **slider** para cambiar la cantidad de círculos.
* Observa los diferentes comportamientos en cada canvas:

  * Movimiento
  * Colisión
  * Rebote
